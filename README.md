Este repositorio contiene diferentes ejercicios, prácticas y desarrollos realizados durante el curso de Sistemas de Información Gerencial.
El propósito principal del repositorio es documentar el proceso de aprendizaje relacionado con la lógica de programación y el desarrollo básico de páginas web, habilidades fundamentales para la creación y gestión de sistemas de información dentro de las organizaciones.
A lo largo del curso se desarrollan distintos ejemplos de código que permiten comprender cómo funcionan las aplicaciones web y cómo la programación puede utilizarse para estructurar, automatizar y gestionar información en entornos digitales.
Este repositorio funciona como un registro organizado de prácticas y proyectos académicos relacionados con la programación y el desarrollo web.
