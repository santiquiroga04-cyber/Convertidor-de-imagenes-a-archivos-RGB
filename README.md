Sistemas de Información Gerencial – Pixel Programming

Hola, soy Santiago Quiroga, estudiante universitario interesado en la programación y los sistemas de información.

Este repositorio documenta mi proceso de aprendizaje en el curso de Sistemas de Información Gerencial, enfocado en el desarrollo de habilidades de lógica de programación a través de Pixel Programming.

A través de estos ejercicios, estoy fortaleciendo mi capacidad para pensar de forma estructurada, resolver problemas y traducir instrucciones en código funcional mediante la construcción de soluciones visuales basadas en píxeles.

Este repositorio contiene ejercicios, prácticas y desarrollos realizados utilizando Pixel Programming, una metodología que permite aprender programación mediante la manipulación de elementos visuales (píxeles) en un entorno estructurado.

El enfoque principal es desarrollar la lógica computacional, entendiendo cómo una serie de instrucciones simples pueden generar resultados complejos a nivel visual.

Cada ejercicio representa una solución a un problema específico, donde se aplican conceptos fundamentales de programación para construir figuras, patrones o comportamientos dentro de una cuadrícula de píxeles.

Objetivos del Repositorio
Desarrollar habilidades de lógica de programación
Comprender cómo funcionan las instrucciones en un programa
Aprender a resolver problemas mediante pensamiento algorítmico
Construir soluciones visuales utilizando píxeles
Fortalecer la capacidad de estructurar código de manera clara y eficiente

1. Entorno de Desarrollo

El desarrollo de los ejercicios se realiza utilizando herramientas accesibles que facilitan el aprendizaje y la experimentación:

Google Colab: utilizado para escribir, ejecutar y probar código de forma interactiva
Microsoft Excel: utilizado como apoyo para estructurar lógica, organizar datos y visualizar patrones

Estas herramientas permiten combinar el pensamiento lógico con la visualización, facilitando el aprendizaje de programación.

Propósito Académico

Este repositorio tiene un enfoque educativo, centrado en el aprendizaje progresivo de la programación a través de Pixel Programming.

El objetivo es construir una base sólida en lógica computacional, la cual es esencial para el desarrollo de sistemas más complejos en el futuro, incluyendo aplicaciones web y sistemas de información empresariales.

REFLEXIÓN

El aprendizaje de programación a través de Pixel Programming me ha permitido entender que programar no se trata solo de escribir código, sino de desarrollar una forma de pensar.

A lo largo de los ejercicios, he aprendido a descomponer problemas, identificar patrones y construir soluciones paso a paso, entendiendo que incluso las tareas más complejas pueden resolverse a partir de instrucciones simples y bien estructuradas.

Este proceso ha fortalecido mi pensamiento lógico y mi capacidad de análisis, sentando una base sólida para enfrentar retos más avanzados en el desarrollo de sistemas de información.
